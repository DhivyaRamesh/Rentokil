No Zones Exist!

Before you can start tracking assets in their locations, you must firstly set up one or more zones. A zone is a short name given to a location that will automatically count the number of assets inside it. Assigning high and low stock levels to a zone will dynamically alter the visual feedback and trends you need in order to make better informed decisions about your assets.

Once you have configured your zones, you will need to start tagging your assets so that they can be tracked inside of them. For each asset you wish to track you must attach a tag to it. Once this is done, you finally need to add the tag including a description of the asset to the system so that you can search for and locate the asset when in use at a later date.
